# 制造业流程

## 需求计划
1. 集团下发 18个月滚动需求预测
2. 工厂做分厂需求预测
3. 线下收集新品/改版需求（邮件/会议纪要）
4. 输出：需求基线 → 下发生产计划

## 生产计划
1. 按周期：
    • ＞6个月
    • 3周～6个月
    • 3周内
2. 考虑：产能、人力、设备、换线、物料齐套、质量约束
3. 下达：生产订单 + 人员排班
4. 输出：生产执行指令 → 同步采购计划

## 采购计划
1. 生产计划 → 拆解为物料采购计划
2. 合同申请
3. 每周四采购下单
4. 控制新旧版物料库存
5. 输出：供应商交付计划 → 材料日到货计划

## 原料到货仓储
1. 供应商到货 → 收货
2. 分行收货 → 手动合托 → 补打ID标签
3. 原包材检验
4. 质量放行 / 暂扣 / 退货
5. 入库上架：先入先出、新旧版区分、批次管理

## 物料配送至产线
1. 生产线下领料单
2. 仓库齐套检查
3. 按生产节拍发料、配送
4. 产线备料、签收确认
5. 处理尾数、退料、换线物料

## 生产作业主线
1. 配方参数下达
2. 开线/换线准备
3. 工艺
4. 关键设备数据采集、参数监控
5. 过程产量、效率、损耗、停机记录
6. 产出下线 → 待检成品

## 生产全流程质量管控
### 原包材质量
    • 标准确定→ 监控计划→ 检验 → 偏差处理 → 供应商改善
### 生产过程质量
    • HACCP评估 → 在线检测→ 异物/缺陷控制 → 偏差跟踪
### 成品质量
    • 成品标准 → 检验→偏差审批 → 质量放行 → 报告存档
### 投诉与追溯
    • 消费者投诉→ 线下分析 → 追溯 → 整改 →审批

## 成品入库
1. 包装下线 → 成品检验 → 质量放行
2. 成品入库
3. 库位管理、批次管理
4. 成品库存监控

## 成品发货物流
1. 接收分仓需求
2. 制定日发货计划
3. 承运商平台叫车
4. 备货、拣货、装车（按SKU顺序/满车发运）
5. 在途追踪 → 交付经销商/客户
6. 绿色发货、车辆利用率考核

## 生产运营分析
1. 数据采集
2. 生成日报/周报/月报
3. 看板展示
4. 效率、成本、质量、能耗分析
5. 输出：改善行动 → 反哺下一轮计划

## 并行支撑业务 · 全程同步运行

### 设备全生命周期管理
#### 设备导入
    • 线下项目交接 →建固定资产 → 搭建资产架构
#### 设备维保
    • 维护计划 → 工单排程 → 执行确认 → 备件领用/采购 → 故障分析
#### 设备备件
    • 库存监控 → 备件申请→ 审批 →采购 → 收货
#### 设备退出
    • 处置申请→ 审批 → 资产处置 → 实物处置

### 能源管理
1. 水/电/气/蒸汽点位采集）
2. 总表/车间表/设备监控
3. 能耗统计、分析、评价
4. 锅炉/制冷/空压机效率优化

### 安环管理
1. 机械安全/特种设备/职业健康/危废管理
2. 承包商管理、高风险作业
3. 有限空间作业（向应急管理局报备）
4. 变更管理、环境合规

### 人力资源管理
1. 人力规划 → 预算 → 招聘 → 入职
2. 培训→ 绩效 → 薪酬
3. 排班→ 工时对接 → 人员分析
4. 行政：安保、宿舍、饭堂、保洁等

### 数据管理
1. 多系统数据采集
2. 数据汇总、清洗、加工
3. 指标体系、BI看板、决策支持
4. 数据复盘 → 驱动计划与业务优化
