# 看见概率

> 布朗大学（Brown University）开发的交互式概率论可视化学习工具。

## 关于看见概率

[看见概率（Seeing Theory）](https://seeing-theory.brown.edu/cn.html) 是一个免费的在线概率论学习平台，通过交互式可视化帮助理解抽象的概率概念。

## 内容涵盖

看见概率包含以下核心主题：

1. **基础概率** - 事件、概率空间、条件概率
2. **随机变量** - 离散与连续随机变量、期望值
3. **概率分布** - 常见分布及其性质
4. **统计推断** - 估计、假设检验
5. **回归分析** - 线性回归、相关性
